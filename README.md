OT Forum Benchmarking App
Self-scoring maturity assessment for the Dragos OT Forum benchmarking session.
About
Attendees score their organization across 5 OT Critical Controls using a mobile-friendly survey. Results post anonymously to SharePoint via Power Automate. The projector dashboard displays live aggregated results for the room.
Controls Covered

CC01 - OT-Specific Incident Response Plan
CC02 - Defensible Architecture
CC03 - ICS Network Visibility and Monitoring
CC04 - Secure Remote Access
CC05 - Risk-Based Vulnerability Management
